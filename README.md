# MySNProject

Ce projet a été réalisé dans le cadre de l’épreuve de versioning Git et GitHub. Il met en place un workflow Git collaboratif incluant la gestion de branches, les Pull Requests, les Issues et une automatisation avec GitHub Actions.  

## 📌 Fonctionnalités  
- Initialisation et configuration d’un dépôt Git.  
- Création d’un dépôt distant sur GitHub et liaison avec le local.  
- Gestion des branches et Pull Requests.  
- Utilisation des Issues pour le suivi des améliorations.  
- Mise en place d’une simple automatisation via GitHub Actions.
